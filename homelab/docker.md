# Docker

## Overview

Docker provides application isolation through lightweight containers, allowing infrastructure services to be deployed, updated, and managed independently.

Containerization simplifies maintenance while reducing resource consumption compared to traditional virtual machines.

---

# Objectives

- Simplify application deployment
- Isolate services
- Improve scalability
- Reduce maintenance complexity
- Support rapid experimentation
