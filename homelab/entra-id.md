# Microsoft Entra ID

## Overview

Microsoft Entra ID provides cloud identity services for authentication, authorization, and administrative access throughout the lab.

Identity is a core security boundary, making Entra ID one of the most important technologies in the environment.

---

# Objectives

- Learn cloud identity management
- Implement least privilege
- Secure privileged accounts
- Explore Conditional Access
- Improve identity governance

---

# Current Features

- User Administration
- Group Management
- Role-Based Access Control
- Multi-Factor Authentication
- Authentication Methods
- Microsoft 365 Integration

---

# Security Principles

The identity platform is built around:

- Least Privilege
- Zero Trust concepts
- Strong Authentication
- Role Separation
- Centralized Identity

---

# Future Enhancements

- Hybrid Identity
- Entra Connect
- Conditional Access Policies
- Identity Protection
- Privileged Identity Management (PIM)

---

# Lessons Learned

Identity is often the first target during an attack. Securing authentication, administrative roles, and user access provides one of the greatest opportunities to reduce organizational risk.
