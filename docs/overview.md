# Enterprise Security Lab Overview

## Purpose

This lab was built to simulate an enterprise environment where I can safely develop practical cybersecurity skills while protecting my own home infrastructure.

Rather than focusing on individual technologies, I designed this environment to better understand how identity, networking, monitoring, endpoint security, and infrastructure work together to reduce organizational risk.

The lab allows me to test new ideas, validate configurations, practice troubleshooting, and document lessons learned before applying similar concepts professionally.

---

## Goals

- Strengthen Identity & Access Management skills
- Improve enterprise networking knowledge
- Practice Security Operations and monitoring
- Gain hands-on SIEM experience
- Build repeatable documentation
- Continue learning through experimentation

---
