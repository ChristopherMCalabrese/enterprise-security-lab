# enterprise-security-lab
Hands-on cybersecurity lab focused on Identity &amp; Access Management, Microsoft 365, network security, detection engineering, and continuous learning.

I built this lab because I learn best by building, testing, breaking, and fixing technology myself.

Throughout my career I've often been the person customers and engineers turned to during difficult technical problems. Building this environment allows me to deepen my understanding of enterprise security technologies, validate new ideas before recommending them professionally, and continue developing the practical skills needed to better protect organizations.

My goal isn't simply to learn products—it's to better understand how security decisions affect people, business operations, and risk.
